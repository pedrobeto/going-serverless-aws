- Amazon DynamoDB
- Amazon RDS
- Amazon Aurora
- Amazon S3
- Amazon IAM
- Amazon SQS
- Amazon Rekognition
- AWS Systems Manager Parameter Store
- Amazon API Gateway
- Amazon EC2 Container Service
- AWS Fargate
